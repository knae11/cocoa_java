# 초간단 RPG

## 기능
*똑같이 구현하지 않아도 되니 재미있는 게임을 만들어 보자.*    
- 11 * 11 배열 화면 중앙에는 캐릭터 위치   
- 임의의 장소에 몬스터, 지뢰 위치 (단 지뢰는 죽기전까지 화면에 미표시)  
- 콘솔 입력으로 WASD로 상좌하우로 이동함   
- 캐릭터가 지뢰를 밟으면 죽음   
- 캐릭터가 몬스터 만나면 이기고, 점수 획득
- 이기면 새로운 게임 시작. 지뢰의 갯수는 2배 증가

## 학습
1. 기능먼저 구현
2. 객체, 함수로 리팩토링
