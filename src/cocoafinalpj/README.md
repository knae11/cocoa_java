### 피카츄 아닌 피카츄 배구 만들기

1. 프로젝트 창에 배경을 넣어 띄운다.
2. 캐릭터 2개를 생성한다.
3. 네트를 설정한다.
4. 캐릭터 마다 움직일 수 있는 공간과 방향키를 설정한다.
    - 1p
    ` 왼쪽 : a, 위 : w, 아래 : s, 오른쪽 : d, 엔터 : shift`
    - 2p
    ` 방향키, 엔터`
5. 방향키마다 상태를 다르게 표현한다.
6. 공을 만들어서 캐릭터가 치는 방향으로 움직이게 한다.
7. 엔터키로 치면 빠르게 움직인다.
8. 바닥에 공이 닿으면 게임이 끝난다.
9. 이긴 캐릭터가 커지면서 기뻐하고, 진 캐릭터는 작아지면서 슬퍼한다.
